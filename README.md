# Virtual Tree

An isometric web game where players plan virtual trees, track CO2 scores, and compete on leaderboards.

## 🎮 Game Features

- **Plant Trees**: Click anywhere on the terrain to plant a tree
- **Track CO₂**: Watch your CO₂ score accumulate as trees grow
- **Persistent State**: Your forest is saved in browser LocalStorage
- **Scrollable Terrain**: Large map with smooth camera movement
- **Player Names**: Tag your trees with your name

## 🚀 Getting Started

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Play Online

The game is deployed to GitHub Pages: [Play Virtual Tree](https://[your-username].github.io/virtual-tree/)

## 🛠️ Tech Stack

- **Phaser 3** - Game framework
- **TypeScript** - Type-safe development
- **Vite** - Build tool and dev server
- **LocalStorage** - Browser-based persistence

## 📝 Controls

- **Click** - Plant a tree
- **Arrow Keys** - Move camera
- **Space** - Change player name
