# F-0014: One Tree Per Cell Constraint — Acceptance Criteria

## Context
Planting two trees on the same grid cell causes visual overlap and makes individual trees indistinguishable.

## Acceptance Criteria

### AC-1: Prevention
- [ ] Clicking a cell that already contains a tree does nothing (no tree placed, no error, no sound)
- [ ] Constraint applies regardless of species — you cannot stack different species on the same cell

### AC-2: Persistence
- [ ] Loaded game state cannot result in two trees sharing a cell (guards against stale data)
