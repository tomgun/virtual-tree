# F-0016 · Forest Animals — Acceptance Criteria

## Feature
Animated critters wander the isometric forest and occlude behind trees.

## Acceptance Checklist

### Population
- [ ] 0 animals when fewer than 10 trees exist
- [ ] 1 animal appears when the 10th tree is planted
- [ ] Population grows by 1 for every additional 10 trees

### Species
- [ ] Three species present: mouse, ant, beetle (bug)
- [ ] Each is drawn with a distinct programmatic shape and colour
- [ ] Small shadow ellipse beneath each creature

### Behaviour
- [ ] Animals start in *idle* state and pause for 0.8–4 s
- [ ] From idle, they transition to *wander* (slow, ~30–60 px/s) or *dash* (fast, 80–220 px/s toward a nearby tree)
- [ ] Sprite flips horizontally to face direction of travel
- [ ] Wandering animals show a subtle vertical bob

### Depth / Occlusion
- [ ] Animal depth is recomputed each frame from world position using the same `IsometricUtils.getDepth()` formula as trees
- [ ] An animal moving to a tree's position renders behind the tree's trunk/foliage when it is at the same or smaller depth value

### Performance
- [ ] 10 animals active simultaneously causes no visible frame-rate drop on a mid-range laptop
