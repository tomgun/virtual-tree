# F-0010: Mobile / Touch Support — Acceptance Criteria

## Context
Players on phones and tablets should be able to play the game with touch controls.

## Acceptance Criteria

### AC-1: Touch-to-plant
- [ ] Single tap on the terrain plants a tree at the tapped grid cell
- [ ] Toolbar buttons respond to tap

### AC-2: Pinch-to-zoom (stretch goal)
- [ ] Two-finger pinch zooms the camera in/out within defined min/max bounds

### AC-3: Drag to pan
- [ ] Single-finger drag scrolls the camera (no tree planted while dragging)
- [ ] Drag is distinguished from tap by a minimum distance threshold (e.g. 10 px)

### AC-4: Layout
- [ ] All UI elements fit within a 375×667 (iPhone SE) viewport without overlapping
- [ ] Toolbar button tap targets are at least 44×44 px

### AC-5: Minimap on mobile
- [ ] Minimap is accessible; optionally hidden behind a toggle to free screen space
