# F-0005: CO2 Scoring System — Acceptance Criteria

## Context
Each tree absorbs CO₂ as it grows. The game must aggregate and display this score to motivate planting.

## Acceptance Criteria

### AC-1: Per-tree CO₂ calculation
- [ ] Each species has a defined `co2PerYear` rate (kg/year)
- [ ] A tree's CO₂ contribution scales linearly with its current age relative to max age
- [ ] `CO2Calculator.calculateForTree` returns 0 for age 0 and `co2PerYear` for a fully grown tree

### AC-2: Total score display
- [ ] Score shown in top-left corner, updated at least every second
- [ ] Score formatted with units: values < 1000 kg shown as "X kg", ≥ 1000 as "X.X t"

### AC-3: Score accuracy
- [ ] Total score equals the sum of all individual tree contributions
- [ ] Score recalculates correctly after page refresh using persisted data
