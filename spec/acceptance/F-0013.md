# F-0013: Age Label Toggle — Acceptance Criteria

## Context
Age labels above trees are useful for debugging but can clutter the view. Players can hide them.

## Acceptance Criteria

### AC-1: Toggle behaviour
- [ ] Press `A` to hide all age labels; press `A` again to show them
- [ ] Labels are visible by default on first load
- [ ] Toggle works at any time when no dialog is open

### AC-2: Consistency
- [ ] Trees planted while labels are hidden start with their label hidden
- [ ] Trees loaded from LocalStorage on page refresh respect the current toggle state
- [ ] Age labels update their text correctly whether visible or hidden (hidden labels still tick)
