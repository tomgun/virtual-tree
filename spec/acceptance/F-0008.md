# F-0008: Minimap Navigation — Acceptance Criteria

## Context
A minimap in the top-right corner gives players an overview of the world and lets them jump to any area.

## Acceptance Criteria

### AC-1: Overview
- [ ] Minimap shows a scaled-down representation of the full world in the top-right corner
- [ ] Tree positions are rendered as coloured dots on the minimap
- [ ] Minimap viewport rectangle shows the current camera view area

### AC-2: Navigation
- [ ] Clicking anywhere on the minimap pans the main camera to the corresponding world position
- [ ] Minimap clicks do NOT plant a tree (blocked at the input level)

### AC-3: Updates
- [ ] Minimap refreshes at least every 500 ms to reflect newly placed or grown trees

### AC-4: Layout
- [ ] Minimap does not overlap the CO₂ score or player name HUD elements
- [ ] Minimap repositions correctly when the browser window is resized
