# F-0012: Tree Species Selector — Acceptance Criteria

## Context
Players choose which species to plant before clicking the terrain. Selection persists until changed.

## Acceptance Criteria

### AC-1: Toolbar
- [ ] A toolbar at the bottom of the screen shows one button per species (Oak, Pine, Palm, Cherry, Birch)
- [ ] Each button displays: species emoji, key shortcut number, species name, and CO₂ rate
- [ ] The currently selected species button is visually highlighted (green border + dark background)
- [ ] Hovering a button shows a tooltip with the species description
- [ ] Toolbar remains horizontally centred after window resize

### AC-2: Keyboard shortcuts
- [ ] Keys `1`–`5` select the corresponding species (Oak=1, Pine=2, Palm=3, Cherry=4, Birch=5)
- [ ] Shortcuts work at any time when no dialog is open

### AC-3: Hit detection
- [ ] Toolbar clicks use screen-space coordinates — selection works correctly regardless of camera position
- [ ] Clicking a toolbar button does NOT plant a tree
